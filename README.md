# MOL_LAB DIDACTIC

**MOL_LAB DIDACTIC** ist eine didaktisch reduzierte Molekülwerkstatt für den Chemieunterricht der SEK2.  
Das Notebook verbindet einfache Moleküldarstellung, RDKit-Deskriptoren, Funktionsgruppen-Erkennung, Konformere und kurze Lernfragen.

[![In Colab öffnen](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/ekerzendorfer/MOL_LAB_DIDACTIC/blob/main/MOL_LAB_DIDACTIC_v0_1_3.ipynb)

## Version 0.1.3

Änderungen gegenüber v0.1.2:

- 2D-Struktur und 3D-Modell werden nebeneinander angezeigt.
- Die Eigenschaftentabelle rückt dadurch weiter nach oben.
- Die 3D-Darstellung kann zwischen Kugel-Stab, Stabmodell und Kalottenmodell umgeschaltet werden.
- Konformere werden nebeneinander in getrennten, unabhängig drehbaren 3D-Fenstern angezeigt.
- Bei kleinen Molekülen wird deutlicher erklärt, warum oft nur wenige echte Konformer-Minima übrig bleiben.

## Start

1. Dateien ins Root-Verzeichnis des GitHub-Repos legen.
2. Notebook über den Colab-Button öffnen.
3. In Colab: **Laufzeit → Alle ausführen**.
4. Modul und Molekül auswählen.
5. Analyse starten.

## Ziel

Nicht maximale Forschungsnähe, sondern ein schülernaher Zugang zur Leitfrage:

> Wie kann man aus Molekülstruktur, funktionellen Gruppen, Polarität und Konformationen auf Stoffeigenschaften schließen?

## Module

- Polarität und Löslichkeit
- Isomerie
- Funktionelle Gruppen
- Konformere
- Wirkstoffe und Biomoleküle

## Abgrenzung zu MOL_LAB

Das große MOL_LAB bleibt das fortgeschrittene Werkzeug mit xTB und erweiterten Berechnungen.  
MOL_LAB DIDACTIC ist bewusst einfacher und für Unterrichtssituationen optimiert.
